📱 QR Code API Documentation

Base URL: https://api.gadgethumans.com/

Generate QR codes from any text or URL. Supports PNG, SVG, and GIF output. No API key required for the free tier.

GET https://api.gadgethumans.com/

Query Parameters

ParameterTypeDefaultDescription
textstringrequiredThe text or URL to encode in the QR code
formatstringpngOutput format: png, svg, or gif
sizenumber200Image size in pixels (100-1000)
colorhex000000Foreground color (e.g. ff0000 for red)
bgcolorhexffffffBackground color
eccstringMError correction: L (7%), M (15%), Q (25%), H (30%)
marginnumber4Quiet zone margin in modules

Examples

Generate a PNG QR Code

curl "https://api.gadgethumans.com/?text=https://example.com" > qr.png

Generate an SVG QR Code (for responsive web use)

curl "https://api.gadgethumans.com/?text=https://example.com&format=svg" > qr.svg

Custom Colors & Size

curl "https://api.gadgethumans.com/?text=https://gadgethumans.com&size=500&color=0070f3&bgcolor=f0f0f0" > custom.png

GIF Output

curl "https://api.gadgethumans.com/?text=https://example.com&format=gif" > qr.gif

HTML Image Tag

<img src="https://api.gadgethumans.com/?text=https://example.com&size=300" alt="QR Code">

JavaScript Fetch

// Get SVG
const response = await fetch('https://api.gadgethumans.com/?text=https://example.com&format=svg');
const svg = await response.text();
document.getElementById('qr').innerHTML = svg;

// Get PNG as blob
const imgResponse = await fetch('https://api.gadgethumans.com/?text=https://example.com');
const blob = await imgResponse.blob();
const imgUrl = URL.createObjectURL(blob);

Python

import requests
resp = requests.get('https://api.gadgethumans.com/', params={
    'text': 'https://example.com',
    'size': 300,
    'format': 'png'
})
with open('qr.png', 'wb') as f:
    f.write(resp.content)

Headers

All responses include:

  • X-RateLimit-Limit — Daily request limit
  • X-RateLimit-Remaining — Remaining requests today
  • X-RateLimit-Reset — Unix timestamp when limit resets

Authentication (Pro/Business)

Subscribe via the pricing page to get an API key. Then include it in requests:

curl "https://api.gadgethumans.com/?text=https://example.com&api_key=your_key_here"
