🔐 Hash Generator API Documentation

Base URL: https://api.gadgethumans.com/

Generate cryptographic hashes for any text using MD5, SHA-1, SHA-256, or SHA-512 algorithms. Supports hex and base64 encoding. No API key required for the free tier.

GET https://api.gadgethumans.com/hash

Query Parameters

ParameterTypeDefaultDescription
textstringrequiredThe text to hash
algorithmstringsha256Hash algorithm: md5, sha1, sha256, or sha512
encodingstringhexOutput encoding: hex or base64

Examples

SHA-256 Hash (Default)

curl "https://api.gadgethumans.com/hash?text=hello+world"
# 2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824

MD5 Hash

curl "https://api.gadgethumans.com/hash?text=hello+world&algorithm=md5"
# 5eb63bbbe01eeed093cb22bb8f5acdc3

SHA-512 Hash

curl "https://api.gadgethumans.com/hash?text=hello+world&algorithm=sha512"
# 309ecc489c12d6eb4cc40f50c902f2b4d0ed77ee511a7c7a9bcd3ca86d4cd86f989dd35bc5ff499670da34255b45b0cfd830e81f605dcf7dc5542e93ae9cd76f

Base64 Encoding

curl "https://api.gadgethumans.com/hash?text=hello+world&algorithm=sha256&encoding=base64"
# LPIk2l+wow4m6DsqxbninhsWHlwfp0JecwQzYpOLmCQ=

JavaScript (Fetch)

const response = await fetch('https://api.gadgethumans.com/hash?text=hello&algorithm=sha256');
const hash = await response.text();
console.log(hash);
// 2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824

Python

import requests
resp = requests.get('https://api.gadgethumans.com/hash', params={
    'text': 'hello world',
    'algorithm': 'sha256',
    'encoding': 'hex'
})
hash_value = resp.text
print(hash_value)

Supported Algorithms

AlgorithmHash LengthUse Case
md5128 bits (32 hex chars)Checksums, legacy systems (not cryptographically secure)
sha1160 bits (40 hex chars)Git commits, legacy verification (deprecated for security)
sha256256 bits (64 hex chars)General purpose, certificates, blockchain (recommended)
sha512512 bits (128 hex chars)High-security applications, password hashing

Headers

All responses include:

  • X-RateLimit-Limit — Daily request limit
  • X-RateLimit-Remaining — Remaining requests today
  • X-RateLimit-Reset — Unix timestamp when limit resets
