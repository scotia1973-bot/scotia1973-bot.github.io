✅ Email Verification API Documentation

Base URL: https://email-verify.scotia1973.workers.dev/

Real-time email validation. Checks format, DNS MX records, and detects disposable email addresses. No API key required for free tier.

POST https://email-verify.scotia1973.workers.dev/verify

Request Body

{
  "email": "user@example.com"
}

Response

{
  "email": "user@example.com",
  "valid_format": true,
  "has_mx": true,
  "is_disposable": false,
  "is_role_based": false,
  "score": 95,
  "reason": "valid"
}

Response Fields

FieldTypeDescription
emailstringThe email address that was checked
valid_formatbooleanWhether the email format is RFC-compliant
has_mxbooleanWhether the domain has mail exchange (MX) records
is_disposablebooleanWhether it's a temporary/disposable email domain
is_role_basedbooleanWhether it's a role-based address (admin@, info@)
scorenumberConfidence score 0-100
reasonstringHuman-readable result: valid, invalid, risky, etc.

Examples

cURL

curl -X POST https://email-verify.scotia1973.workers.dev/verify \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com"}'

JavaScript (Fetch)

const response = await fetch('https://email-verify.scotia1973.workers.dev/verify', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ email: 'user@example.com' })
});
const result = await response.json();
console.log(result.score); // 95
console.log(result.valid_format); // true

Python

import requests
resp = requests.post('https://email-verify.scotia1973.workers.dev/verify', 
    json={'email': 'user@example.com'})
result = resp.json()
print(f"Score: {result['score']}/100")
print(f"Valid: {result['valid_format'] and result['has_mx']}")
