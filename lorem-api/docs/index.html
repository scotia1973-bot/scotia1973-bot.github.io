📝 Lorem Ipsum Generator API Documentation

Base URL: https://api.gadgethumans.com/

Generate Lorem Ipsum placeholder text for wireframes, mockups, and UI development. Choose between paragraphs, sentences, or words. No API key required for the free tier.

GET https://api.gadgethumans.com/lorem

Query Parameters

ParameterTypeDefaultDescription
typestringparagraphsOutput type: paragraphs, sentences, or words
countnumber3Number of units to generate (1-100)

Examples

Generate 3 Paragraphs (Default)

curl "https://api.gadgethumans.com/lorem"

Generate 5 Sentences

curl "https://api.gadgethumans.com/lorem?type=sentences&count=5"

Generate 10 Words

curl "https://api.gadgethumans.com/lorem?type=words&count=10"
# Lorem ipsum dolor sit amet consectetur adipiscing elit

Single Paragraph for a Card Component

curl "https://api.gadgethumans.com/lorem?type=paragraphs&count=1"

JavaScript (Fetch)

const response = await fetch('https://api.gadgethumans.com/lorem?type=sentences&count=3');
const text = await response.text();
console.log(text);
// Lorem ipsum dolor sit amet... [3 sentences]

Python

import requests
resp = requests.get('https://api.gadgethumans.com/lorem', params={
    'type': 'paragraphs',
    'count': 5
})
placeholder_text = resp.text
print(placeholder_text)
