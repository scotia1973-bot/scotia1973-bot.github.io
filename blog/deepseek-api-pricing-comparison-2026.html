    Real Cost Calculation: Running 100,000 Conversations Per Month

    Assume each conversation uses **500 input tokens + 200 output tokens**. That’s 700 tokens per conversation. For 100,000 conversations:

  • Total input tokens: 50,000,000 (50M)
  • Total output tokens: 20,000,000 (20M)

    • Let’s compute cost for each model using a Python script.

    import requests

models = {
    "DeepSeek V4 Flash": {"input": 0.015, "output": 0.10},
    "GPT-4o": {"input": 2.50, "output": 10.00},
    "Claude Sonnet 4": {"input": 3.00, "output": 15.00},
    "Gemini 2.5 Pro": {"input": 1.25, "output": 5.00}
}

convs = 100000
input_tokens = 500
output_tokens = 200

for model, rates in models.items():
    cost = (input_tokens * rates['input'] + output_tokens * rates['output']) * convs / 1_000_000
    print(f"{model}: ${cost:.2f}")
