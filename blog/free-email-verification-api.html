Free Email Verification API: Validate Emails in Real-Time (No API Key)

Every form on the internet collects email addresses. But how many of those emails are real?

If you're building a signup form, newsletter, or any application that collects emails, you need to validate them. Bounce rates hurt your sender reputation. Disposable emails waste your resources.

That's why I built a free email verification API — it validates emails in real-time without requiring an API key.

Quick Start

curl -X POST https://email-verify.scotia1973.workers.dev/verify \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com"}'

Response:

{
  "email": "user@example.com",
  "valid_format": true,
  "has_mx": true,
  "is_disposable": false,
  "is_role_based": false,
  "score": 95,
  "reason": "valid"
}

Features

  • Format validation: Checks RFC-compliant email format
  • DNS MX lookup: Verifies the domain can receive mail
  • Disposable detection: Blocks 200+ temporary email domains
  • Role-based detection: Alerts on admin@, info@, support@
  • Scoring: 0-100 confidence score per email
  • Free tier: 50 verifications/day, no API key needed

Why Validate Emails?

  • Reduce bounce rates: Invalid emails hurt your sender reputation
  • Save money: Don't pay for email delivery to invalid addresses
  • Block spam: Disposable emails are often used for abuse
  • Improve data quality: Clean email lists = better conversion rates

  • Lead generation: Ensure lead quality at point of capture
