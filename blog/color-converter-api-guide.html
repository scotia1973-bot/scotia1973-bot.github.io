2. Understanding HEX and RGB

HEX (Hexadecimal)

The HEX color format represents colors as a six-digit (or three-digit shorthand) hexadecimal number prefixed with #. Each pair of digits represents one channel: red, green, and blue. For example, #FF0000 is pure red, #00FF00 is pure green, and #0000FF is pure blue.

HEX can also be written in 3-digit shorthand: #F00 is equivalent to #FF0000. The API supports both 3-digit and 6-digit hex formats.

RGB (Red, Green, Blue)

The RGB format uses decimal values (0–255) to specify each channel. Pure red in RGB is rgb(255, 0, 0), white is rgb(255, 255, 255), and black is rgb(0, 0, 0).

3. The Color Converter API

GET https://api.gadgethumans.com/color?value={color}&from={hex|rgb}&to={hex|rgb}

Parameters

ParameterRequiredDescription
valueYesThe color value to convert. HEX: FF0000 or #FF0000. RGB: rgb(255,0,0) or 255,0,0.
fromNo (default: hex)Source format: hex or rgb
toNo (default: rgb)Target format: hex or rgb

Example Response

{
  "input": {
    "format": "hex",
    "value": "#ff0000"
  },
  "output": {
    "format": "rgb",
    "value": "rgb(255, 0, 0)"
  },
  "components": {
    "r": 255,
    "g": 0,
    "b": 0
  }
}

The response includes both the formatted output string AND individual RGB components — useful for programmatic manipulation.

5. Python Integration

import requests

def convert_color(value, from_format="hex", to_format="rgb"):
    """
    Convert a color between HEX and RGB formats.
    
    Args:
        value: Color value (e.g., "#FF0000" or "rgb(255,0,0)")
        from_format: "hex" or "rgb"
        to_format: "hex" or "rgb"
    
    Returns:
        Dict with input, output, and components
    """
    response = requests.get(
        "https://api.gadgethumans.com/color",
        params={"value": value, "from": from_format, "to": to_format}
    )
    
    if response.status_code == 200:
        return response.json()
    else:
        raise Exception(f"API error {response.status_code}: {response.text}")

# Convert GadgetHumans brand blue to RGB
result = convert_color("#4a90d9", "hex", "rgb")
print(f"HEX #4a90d9 → {result['output']['value']}")
print(f"Components: R={result['components']['r']}, "
      f"G={result['components']['g']}, B={result['components']['b']}")

# Convert RGB back to HEX
result2 = convert_color("rgb(255, 102, 0)", "rgb", "hex")
print(f"RGB(255,102,0) → {result2['output']['value']}")

6. JavaScript Integration

Node.js

async function convertColor(value, fromFormat = 'hex', toFormat = 'rgb') {
    const params = new URLSearchParams({
        value: value,
        from: fromFormat,
        to: toFormat
    });
    
    const response = await fetch(
        `https://api.gadgethumans.com/color?${params}`
    );
    
    if (!response.ok) {
        throw new Error(`API error: ${response.status}`);
    }
    
    return response.json();
}

// Usage
(async () => {
    // HEX to RGB
    const r1 = await convertColor('#007AFF', 'hex', 'rgb');
    console.log('#007AFF →', r1.output.value);
    
    // RGB to HEX
    const r2 = await convertColor('rgb(255, 59, 48)', 'rgb', 'hex');
    console.log('RGB(255,59,48) →', r2.output.value);
})();

Lighten/Darken Colors

While the API doesn't do lightening/darkening directly, you can combine color conversion with programmatic manipulation:

import requests

def lighten_color(hex_color, factor=0.3):
    """Lighten a hex color by factor (0-1)."""
    # First convert HEX to RGB
    result = requests.get(
        "https://api.gadgethumans.com/color",
        params={"value": hex_color, "from": "hex", "to": "rgb"}
    ).json()
    
    c = result["components"]
    
    # Lighten each channel
    r = min(255, int(c["r"] + (255 - c["r"]) * factor))
    g = min(255, int(c["g"] + (255 - c["g"]) * factor))
    b = min(255, int(c["b"] + (255 - c["b"]) * factor))
    
    # Convert back to HEX
    lighter_rgb = f"rgb({r},{g},{b})"
    result2 = requests.get(
        "https://api.gadgethumans.com/color",
        params={"value": lighter_rgb, "from": "rgb", "to": "hex"}
    ).json()
    
    return result2["output"]["value"]

# Lighten the primary color by 40%
primary_light = lighten_color("#007AFF", 0.4)
print(f"Primary: #007AFF → Lightened: {primary_light}")

CSS Variable Generation

Use the API to generate CSS variables in both HEX and RGB for better color manipulation:

const colors = ['#007AFF', '#5856D6', '#34C759', '#FF9500', '#FF3B30'];

async function generateCSSVariables(hexColors) {
    for (const hex of hexColors) {
        const r = await fetch(
            `https://api.gadgethumans.com/color?value=${encodeURIComponent(hex)}&from=hex&to=rgb`
        ).then(res => res.json());
        
        const { r: red, g: green, b: blue } = r.components;
        
        console.log(`--color-hex: ${hex};`);
        console.log(`--color-rgb: ${red}, ${green}, ${blue};`);
        console.log(`--color: rgb(var(--color-rgb));`);
        console.log(`--color-light: rgba(var(--color-rgb), 0.1);`);
        console.log('---');
    }
}

generateCSSVariables(colors);
