Quick Start

Generate a QR code in one line:

curl "https://api.gadgethumans.com/?text=https://example.com" > qr.png

Or use it in an HTML image tag:

<img src="https://api.gadgethumans.com/?text=https://example.com&size=300" alt="QR Code">
