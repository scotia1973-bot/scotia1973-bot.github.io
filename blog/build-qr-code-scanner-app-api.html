How to Build a QR Code Scanner App with a Free API (Full Tutorial)

Last updated: May 2026 · 12 min read · Intermediate

⏱️ What you'll build: A web-based QR code scanner that uses your device camera to scan QR codes in real time, then generates new QR codes using the GadgetHumans QR API. Full code in ~80 lines of JavaScript.

QR code scanner apps are everywhere — restaurant menus, event check-ins, ticketing, package tracking, contact tracing. Most tutorials show you how to scan QR codes with a phone camera, but they miss the second half: generating QR codes programmatically from the scanned data.

In this tutorial, I'll show you how to build a complete QR code scanner + generator app using:

  • Your device camera + the html5-qrcode library for scanning
  • The GadgetHumans QR Code API (free, 100 codes/day) for generating codes from scanned data
  • Vanilla JavaScript — no frameworks needed

What We're Building

A single HTML page that does three things:

  1. Scans QR codes using your phone or laptop camera
  2. Displays the decoded content (URL, text, or contact info)
  3. Generates a new QR code from the scanned data — or from custom input — using the GadgetHumans API

Step 3: Initialize the QR Scanner

Now let's wire up the camera scanner. The html5-qrcode library makes this trivially simple:

3 Scanner JavaScript

const html5QrCode = new Html5Qrcode("reader");

function startScanner() {
  html5QrCode.start(
    { facingMode: "environment" }, // Use back camera
    { fps: 10, qrbox: { width: 250, height: 250 } },
    onScanSuccess
  ).catch(err => {
    document.getElementById('result').innerHTML =
      '<p>Camera error: ' + err + '</p>';
  });
}

function onScanSuccess(decodedText, decodedResult) {
  // Stop scanning once we get a result
  html5QrCode.stop();

  document.getElementById('result').innerHTML =
    '<h3>✅ Scanned:</h3>' +
    '<p><a href="' + decodedText + '" target="_blank">'
    + decodedText + '</a></p>';

  // Auto-generate a QR code from the scanned data
  generateQR(decodedText);
}

// Start scanning on page load
startScanner();

Step 4: Generate QR Codes with the API

This is where the GadgetHumans QR API shines. With a single GET request, we get back a ready-to-display QR code image:

4 QR Generation Function

function generateQR(text) {
  if (!text) return;

  const encoded = encodeURIComponent(text);
  const qrUrl = `https://api.gadgethumans.com/qr?url=${encoded}&size=400&color=58a6ff&bgcolor=0d1117`;

  document.getElementById('qrcode').innerHTML =
    '<h3>🔄 Generated QR Code:</h3>' +
    '<img src="' + qrUrl + '" alt="QR Code" />' +
    '<p style="font-size:0.8rem;color:#8b949e;">' +
    'Powered by <a href="https://api.gadgethumans.com">GadgetHumans QR API</a></p>';
}

// Handle manual input
document.getElementById('generateBtn').addEventListener('click', () => {
  const text = document.getElementById('customUrl').value;
  if (text) generateQR(text);
});

// Also generate on Enter key
document.getElementById('customUrl').addEventListener('keypress', (e) => {
  if (e.key === 'Enter') {
    document.getElementById('generateBtn').click();
  }
});

Testing Your App

You need to serve this page over HTTPS for the camera API to work. Three quick options:

  • GitHub Pages: Push to any repo and enable Pages — instant HTTPS
  • VS Code Live Server: Works on localhost with HTTPS via ngrok
  • Python one-liner: python3 -m http.server 8000 + ngrok

Once it's running, point your phone's camera at any QR code (or use a second device to display one), and watch the app scan it, display the decoded content, and generate a fresh QR code — all in under a second.

Taking It Further: Ideas for Production

This basic app is functional, but you can extend it in several ways:

  • Batch scanning: Add a file input for uploading screenshots containing QR codes
  • History log: Store scanned codes in localStorage with timestamps
  • Export: Download generated QR codes as PNG or SVG
  • Branding: Use the API's logo parameter to overlay your brand logo
  • Analytics: Track which QR codes get scanned most often
