2. API Endpoint & Parameters

The QR code API accepts GET requests with URL query parameters:

GET https://api.gadgethumans.com/qr?data={text}&format={png|svg|jpg}&size={pixels}
ParameterTypeDefaultDescription
datastring—The content to encode (URL, text, phone number, etc.)
formatstringpngOutput image format: png, svg, or jpg
sizeinteger300Image dimensions (100–2000px)
foregroundstring#000000Hex color for QR modules (e.g., #ff6600)
backgroundstring#FFFFFFHex color for background
eccstringMError correction: L/M/Q/H
margininteger4Quiet zone margin (0–20 modules)

The API returns raw image data (not JSON), so you can pipe it directly to a file or display it in a browser.

4. Python Integration

Python is perfect for batch processing and automated workflows:

import requests

def generate_qr(data, filename="qrcode.png", size=300, format="png"):
    url = "https://api.gadgethumans.com/qr"
    params = {"data": data, "format": format, "size": size}
    
    response = requests.get(url, params=params, stream=True)
    
    if response.status_code == 200:
        with open(filename, "wb") as f:
            f.write(response.content)
        print(f"✅ QR code saved: {filename}")
        return filename
    else:
        raise Exception(f"Error {response.status_code}: {response.text}")

# Generate a QR code for your website
generate_qr("https://www.gadgethumans.com", "site-qr.png", 500)

Custom Branded QR Code

def generate_branded_qr(data, foreground="#0f3460", background="#f8f9fa"):

            f.write(response.content)
        return True
    return False

5. JavaScript & Browser Integration

Node.js with Axios

const axios = require('axios');
const fs = require('fs');

async function generateQR(data, filename = 'qrcode.png', size = 300) {
    const params = { data, format: 'png', size: size.toString() };
    
    const response = await axios.get('https://api.gadgethumans.com/qr', {
        params,
        responseType: 'stream'
    });
    
    const writer = fs.createWriteStream(filename);
    response.data.pipe(writer);
    
    return new Promise((resolve, reject) => {
        writer.on('finish', () => resolve(filename));
        writer.on('error', reject);
    });
}

// Usage
generateQR('https://api.gadgethumans.com', 'api-hub-qr.png', 500)
    .then(f => console.log(`✅ Saved: ${f}`))
    .catch(e => console.error(e));

Browser-Side QR Generator

async function displayQR(data, containerId) {
    const params = new URLSearchParams({
        data: data,
        format: 'png',
        size: 400
    });
    
    const url = `https://api.gadgethumans.com/qr?${params}`;
    
    const img = document.createElement('img');
    img.src = url;
    img.alt = `QR code for ${data}`;
    img.style.maxWidth = '100%';
    
    document.getElementById(containerId).appendChild(img);
}

// On page load, generate a QR code
displayQR('https://www.gadgethumans.com', 'qr-container');

This browser approach requires zero server-side code — perfect for SaaS dashboards, admin panels, or client-facing tools.

6. Customization Options

Color Matching Your Brand

Use the foreground and background parameters to match your brand colors. Ensure sufficient contrast — a dark foreground on a light background is most reliable for scanners.

Error Correction Levels

LevelRecoveryBest For
L7%Maximum data density, no logo overlay
M15%General use (default)
Q25%Small logo overlays, moderate damage tolerance
H30%Large logo overlays, maximum damage tolerance

For branded QR codes with a logo in the center, always use ecc=H (30% error correction).

Size Guidelines

  • Digital display: 200–300px
  • Print materials: 500–1000px
  • Billboards/banners: 1000–2000px (use SVG format)

7. Bulk QR Code Generation

Need to generate hundreds of QR codes? Here's how to batch-process URLs efficiently:

import requests
from concurrent.futures import ThreadPoolExecutor
import csv

def generate_from_csv(csv_file, output_dir="qr_codes"):
    import os
    os.makedirs(output_dir, exist_ok=True)
    
    urls = []
    with open(csv_file, 'r') as f:
        reader = csv.reader(f)
        for row in reader:
            if row:
                urls.append(row[0])
    
    def process_url(url):
        safe_name = url.replace("https://", "").replace("/", "_").replace(".", "-")[:50]
        filename = f"{output_dir}/{safe_name}.png"
        
        response = requests.get(
            "https://api.gadgethumans.com/qr",
            params={"data": url, "format": "png", "size": 400}
        )
        if response.status_code == 200:
            with open(filename, "wb") as f:
                f.write(response.content)
            return filename
        return None
    
    with ThreadPoolExecutor(max_workers=5) as executor:
        results = list(executor.map(process_url, urls))
    
    print(f"Generated {len([r for r in results if r])} QR codes")

# Usage: generate_from_csv("product-urls.csv")
