Color Converter API — Free Hex to RGB Color Conversion

June 7, 2026

Introduction

Color conversion is a common need in CSS, UI design, and data visualization. The GadgetHumans Color Converter API provides instant conversion between hex (#FF0000) and RGB (rgb(255, 0, 0)) formats with 33 million+ possible configurations.

API Endpoint

GET https://api.gadgethumans.com/color

Parameters

ParameterRequiredDefaultDescription
valueYes—Color value: "FF0000", "#FF0000", or "rgb(255,0,0)"
from_formatNohexSource format: hex or rgb
to_formatNorgbTarget format: hex or rgb

Examples

Hex to RGB

curl "https://api.gadgethumans.com/color?value=4287f5&from=hex&to=rgb"
# Returns: {"original": "#4287f5", "result": "rgb(66, 135, 245)", "from": "hex", "to": "rgb"}

RGB to Hex

curl "https://api.gadgethumans.com/color?value=rgb(255,0,0)&from=rgb&to=hex"
# Returns: {"original": "rgb(255,0,0)", "result": "#FF0000", "from": "rgb", "to": "hex"}

With Hash Prefix

curl "https://api.gadgethumans.com/color?value=%23FF0000&from=hex&to=rgb"

33 Million+ Configurations

16,777,216 hex colors × 2 conversion directions = 33,554,432 unique configurations. Every color in the 24-bit spectrum is supported.

MCP Tool: convert_color
