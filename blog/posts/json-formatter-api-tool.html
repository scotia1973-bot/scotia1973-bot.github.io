JSON Formatter API Tool — Prettify, Validate, and Minify JSON

June 6, 2026

Why a JSON Formatter API?

Working with JSON is fundamental to modern development. Whether you're debugging API responses, cleaning configuration files, or processing data for AI agents, a JSON formatter API is essential. The GadgetHumans JSON Processor provides three modes — prettify, validate, and minify — all accessible via a simple GET request with no authentication.

API Endpoint

GET https://api.gadgethumans.com/json

Parameters

ParameterRequiredDefaultOptions
json_stringYes—Any JSON string
modeNoprettifyprettify, validate, minify

Examples

Prettify

curl "https://api.gadgethumans.com/json?json_string=%7B%22name%22%3A%22test%22%2C%22value%22%3A42%7D&mode=prettify"
# Returns formatted JSON with indentation

Validate

curl "https://api.gadgethumans.com/json?json_string=%7B%22invalid&mode=validate"
# Returns {"valid": false, "error": "..."}

Minify

curl "https://api.gadgethumans.com/json?json_string=%7B%22name%22%3A%22test%22%7D&mode=minify"
# Returns compact JSON with no whitespace

MCP Tool: process_json

AI agents using the MCP server can call process_json to handle JSON processing tasks. Available via the Streamable HTTP endpoint at https://api.gadgethumans.com/mcp.

Use Cases

  • Debugging API responses
  • Preparing JSON for config files
  • Validating JSON before deserialization
  • Minifying JSON for storage or transmission
