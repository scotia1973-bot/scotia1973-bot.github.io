Hash API — Free MD5, SHA-256, SHA-512 Cryptographic Hashing

June 7, 2026

Introduction

Cryptographic hashing is fundamental to security — used for password storage, data integrity verification, digital signatures, and checksums. The GadgetHumans Hash API provides four industry-standard algorithms (MD5, SHA-1, SHA-256, SHA-512) with hex or base64 output, all accessible with zero authentication.

API Endpoint

GET https://api.gadgethumans.com/hash

Parameters

ParameterRequiredDefaultOptions
textYes—Any string
algorithmNosha256md5, sha1, sha256, sha512
encodingNohexhex, base64

Examples

SHA-256 Hash

curl "https://api.gadgethumans.com/hash?text=hello+world&algorithm=sha256&encoding=hex"
# Returns SHA-256 hash of "hello world" in hex

SHA-512 Hash in Base64

curl "https://api.gadgethumans.com/hash?text=data+integrity&algorithm=sha512&encoding=base64"

MD5 Hash

curl "https://api.gadgethumans.com/hash?text=quick+check&algorithm=md5"

MCP Tool: hash_text

For AI agents, the hash_text MCP tool provides the same functionality. Install the MCP server via pip install gadgethumans-api-hub-mcp or call the Streamable HTTP endpoint directly.

8 Unique Configurations + Unlimited Text
