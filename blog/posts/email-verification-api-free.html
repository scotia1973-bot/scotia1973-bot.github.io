Email Verification API — Free Email Validation Service

June 7, 2026

Why Email Verification Matters

Invalid email addresses cause deliverability issues, waste resources, and harm sender reputation. The GadgetHumans Email Verification API provides comprehensive validation — checking syntax, domain validity, MX record existence, and disposable email provider detection.

API Endpoint

GET https://api.gadgethumans.com/email-verify

Parameters

ParameterRequiredDescription
emailYesEmail address to verify

Example

curl "https://api.gadgethumans.com/email-verify?email=test@example.com"
# Returns: {"email": "test@example.com", "valid": true, "disposable": false, "domain": "example.com", "syntax_valid": true}

MCP Tool: verify_email

AI agents can use the verify_email MCP tool via the MCP server at https://api.gadgethumans.com/mcp or by installing the PyPI package.

What Gets Checked

  • Syntax: RFC 5322 email format validation
  • Domain: DNS lookup confirms domain exists
  • MX Records: Mail exchange records present
  • Disposable: Detects temporary/throwaway email providers

Use Cases

  • User registration validation
  • Contact list cleaning
  • AI agent form processing
  • Lead qualification
