QR Code API Integration Guide — Free QR Generation

June 7, 2026

Introduction

QR codes are everywhere — from restaurant menus to two-factor authentication. The GadgetHumans QR Code API makes it trivial to generate QR codes programmatically. Available as both a REST endpoint and an MCP server tool (generate_qr_code), it's one of the most versatile tools in our AI tools API collection.

API Endpoint

GET https://api.gadgethumans.com/qr

Parameters

ParameterTypeRequiredDefaultDescription
textstringYes—URL or text to encode
formatstringNopngpng, svg, or gif
sizeintNo300100-1000 pixels
colorstringNo000000Foreground hex (no #)
bgcolorstringNoFFFFFFBackground hex (no #)
eccstringNoML (7%), M (15%), Q (25%), H (30%)
marginintNo40-20 modules

Code Examples

Generate a QR Code (cURL)

curl -o qr.png "https://api.gadgethumans.com/qr?text=https://gadgethumans.com&format=png&size=500&color=4F46E5&bgcolor=FFFFFF&ecc=H&margin=4"

Generate QR Code via MCP (Python)

from gadgethumans_api_hub_mcp import mcp
# Or call via MCP protocol
import requests, json
response = requests.post("https://api.gadgethumans.com/mcp", json={
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
        "name": "generate_qr_code",
        "arguments": {
            "text": "https://example.com",
            "format": "svg",
            "size": 500,
            "color": "FF0000",
            "ecc": "H"
        }
    },
    "id": 1
})
print(response.json())

JavaScript Integration

async function generateQR(text, format = 'png') {
  const url = `https://api.gadgethumans.com/qr?text=${encodeURIComponent(text)}&format=${format}`;
  const response = await fetch(url);
  const blob = await response.blob();
  return URL.createObjectURL(blob);
}
