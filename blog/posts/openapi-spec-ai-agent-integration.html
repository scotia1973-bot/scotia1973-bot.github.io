Why OpenAPI Matters for AI Agents

The OpenAPI Specification (OAS) is the industry standard for describing REST APIs in a machine-readable format. AI agents parse OpenAPI specs to understand available endpoints, parameters, and response schemas. The GadgetHumans API Hub provides a comprehensive OpenAPI 3.1 specification at /openapi.json — the definitive API contract for our 200+ tool collection.

OpenAPI Spec Features

  • Version: OpenAPI 3.1.0 (latest standard)
  • Servers: https://api.gadgethumans.com
  • Endpoints: 11 documented paths with full parameter schemas
  • MCP Extension: x-mcp metadata linking to MCP tools
  • LLM Extension: x-llms linking to llms.txt files
  • ChatGPT Extension: x-openai-plugin for GPT integration
  • Tags: Categorized by function (QR, Security, Data, etc.)
  • Response Schemas: Full JSON Schema for every response
