June 7, 2026

Why a Timestamp Converter API?

Working with dates and timestamps across different systems is a common pain point. The GadgetHumans Timestamp Converter API handles conversions between Unix epoch timestamps and human-readable date formats (ISO 8601, RFC 2822) with a single GET request.

API Endpoint

GET https://api.gadgethumans.com/timestamp

Parameters

ParameterRequiredDefaultOptions
valueYes—Unix timestamp or date string
formatNoisoiso, unix, rfc2822

Examples

Date to Unix Timestamp

curl "https://api.gadgethumans.com/timestamp?value=2025-05-29&format=unix"
# Returns: {"original": "2025-05-29", "converted": 1748476800, "format": "unix"}

Unix Timestamp to ISO 8601

curl "https://api.gadgethumans.com/timestamp?value=1717000000&format=iso"

Date to RFC 2822

curl "https://api.gadgethumans.com/timestamp?value=May+29+2025&format=rfc2822"

MCP Tool: convert_timestamp

AI agents can use the convert_timestamp MCP tool via the MCP server. Install via pip or call the Streamable HTTP endpoint.

Supported Input Formats

  • Unix timestamps (seconds since epoch, e.g., 1717000000)
  • ISO 8601 date strings (e.g., "2025-05-29", "2025-05-29T12:00:00Z")
  • Common date strings (e.g., "May 29, 2025")

Use Cases

  • Database timestamp normalization
  • API response date parsing
  • Log file timestamp conversion
  • AI agent time-aware operations
