What is an MCP Server?

The Model Context Protocol (MCP) is an open standard that lets AI agents connect to external tools and data sources. An MCP server exposes tools that AI agents can discover and call at runtime. The GadgetHumans API Hub MCP server is the largest free collection of utility tools available through this protocol — with 200+ tool configurations across 11 REST endpoints, all wrapped as 12 MCP tools you can install in minutes.

Why This MCP Server Stands Out

Most MCP servers provide one or two tools. The GadgetHumans API Hub delivers 200+ configurations across 11 categories: QR code generation, password generation, UUID generation, cryptographic hashing (MD5, SHA-1, SHA-256, SHA-512), Base64 encode/decode, lorem ipsum generation, JSON processing (prettify/validate/minify), color conversion (hex/RGB), email verification, IP geolocation, and timestamp conversion. This is the definitive MCP tools directory for AI agents that need versatile utility capabilities.

Complete Tool Listing

1. QR Code Generator (MCP: generate_qr_code)

Generate QR codes from any text or URL. Supports PNG, SVG, and GIF output. Configurable size (100-1000px), colors (any hex), error correction (L/M/Q/H up to 30%), and margin (0-20). REST endpoint: GET /qr

2. Password Generator (MCP: generate_password)

Generate cryptographically secure random passwords. Configurable length (4-256), character sets (numbers, symbols, uppercase, lowercase), and batch count (1-50). REST endpoint: GET /password

3. UUID Generator (MCP: generate_uuid)

Generate UUID v4 (random) identifiers. Supports batch generation (1-100 at once) and multiple output formats (standard, JSON, array). REST endpoint: GET /uuid

4. Hash Text (MCP: hash_text)

Generate cryptographic hashes with MD5, SHA-1, SHA-256, or SHA-512. Output in hex or base64 encoding. REST endpoint: GET /hash

5. Base64 Encode/Decode (MCP: encode_base64)

Encode text to Base64 or decode Base64 back to plain text. Essential for data transport. REST endpoint: GET /base64

6. Lorem Ipsum Generator (MCP: generate_lorem_ipsum)

Generate placeholder text for mockups. Configurable by paragraphs, sentences, or words (1-100 units). REST endpoint: GET /lorem

7. JSON Processor (MCP: process_json)

Prettify (indent), validate (check syntax), or minify (compact) JSON strings. REST endpoint: GET /json

8. Color Converter (MCP: convert_color)

Convert colors between hex (#FF0000) and RGB (rgb(255,0,0)) formats. REST endpoint: GET /color

9. Email Verifier (MCP: verify_email)

Verify email address validity — checks syntax, domain, MX records, and disposable providers. REST endpoint: GET /email-verify

10. IP Geolocator (MCP: geolocate_ip)

Get geolocation data for any IP address — city, country, ISP, timezone, coordinates. REST endpoint: GET /ip

11. Timestamp Converter (MCP: convert_timestamp)

Convert between Unix timestamps and date formats (ISO 8601, RFC 2822). REST endpoint: GET /timestamp

12. Health Check (MCP: health)

Check availability of all 11 endpoints. Returns per-endpoint UP/DOWN status. REST endpoint: GET /health
