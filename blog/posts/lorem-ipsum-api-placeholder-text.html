Lorem Ipsum API — Free Placeholder Text Generator

June 7, 2026

Introduction

Placeholder text is essential for mockups, wireframes, and UI prototypes. The GadgetHumans Lorem Ipsum API generates classic Lorem Ipsum dummy text via a simple GET request — with 300 unique configurations across three output types.

API Endpoint

GET https://api.gadgethumans.com/lorem

Parameters

ParameterDefaultOptionsRange
typeparagraphsparagraphs, sentences, words—
count3—1-100

Examples

5 Sentences

curl "https://api.gadgethumans.com/lorem?type=sentences&count=5"
# Returns 5 Lorem Ipsum sentences

2 Paragraphs

curl "https://api.gadgethumans.com/lorem?type=paragraphs&count=2"

10 Words

curl "https://api.gadgethumans.com/lorem?type=words&count=10"

300 Configurations

3 types × 100 count levels = 300 unique configurations. Generate anything from a single word to 100 full paragraphs.

MCP Tool: generate_lorem_ipsum

AI agents using the MCP server can call generate_lorem_ipsum for placeholder text generation in autonomous workflows.
