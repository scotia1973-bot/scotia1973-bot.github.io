What is Streamable HTTP MCP?

The Model Context Protocol (MCP) defines how AI agents discover and call external tools. Streamable HTTP is the transport layer that enables agent-to-server communication over standard HTTP. The GadgetHumans API Hub exposes a Streamable HTTP MCP endpoint at https://api.gadgethumans.com/mcp, giving any MCP-compatible agent access to 200+ tool configurations.

MCP Endpoint Details

  • URL: POST https://api.gadgethumans.com/mcp
  • Protocol: JSON-RPC 2.0
  • Content-Type: application/json
  • Transport: Streamable HTTP (SSE support)
  • Authentication: None
  • Rate Limit: 100 req/day per tool (free)

How Agents Use the MCP Endpoint

Step 1: Tool Discovery

POST https://api.gadgethumans.com/mcp
Content-Type: application/json

{
  "jsonrpc": "2.0",
  "method": "tools/list",
  "id": 1
}

Step 2: Call a Tool

POST https://api.gadgethumans.com/mcp
Content-Type: application/json

{
  "jsonrpc": "2.0",
  "method": "tools/call",
  "params": {
    "name": "generate_password",
    "arguments": {
      "length": 20,
      "count": 3,
      "symbols": true,
      "numbers": true
    }
  },
  "id": 2
}

Step 3: Resource Access (Optional)

POST https://api.gadgethumans.com/mcp
Content-Type: application/json

{
  "jsonrpc": "2.0",
  "method": "resources/list",
  "id": 3
}

Compatible Clients

  • Claude Desktop — Add to mcpServers config
  • OpenAI GPTs — Via ChatGPT plugin (/.well-known/ai-plugin.json)
  • Google A2A Agents — Via agent card (/.well-known/agent-card.json)
  • Custom MCP Clients — Any HTTP client supporting JSON-RPC 2.0
  • Python — pip install gadgethumans-api-hub-mcp

Performance and Reliability

The MCP endpoint runs on serverless infrastructure with typical response times under 200ms. The health tool checks all 11 upstream REST endpoints and returns per-endpoint status. Free tier: 100 calls/day per tool. Pro tier ($5/mo): unlimited.
