IP Geolocation API — Free IP Location Service via REST or MCP

June 7, 2026

Introduction

IP geolocation is essential for analytics, geo-targeting, fraud detection, and content localization. The GadgetHumans IP Geolocation API provides detailed location data for any IPv4 address — with zero authentication required.

API Endpoint

GET https://api.gadgethumans.com/ip

Parameters

ParameterRequiredDescription
ip_addressNoIP to look up. Defaults to caller's IP.

Examples

Geolocate a Specific IP

curl "https://api.gadgethumans.com/ip?ip_address=8.8.8.8"
# Returns: {"ip": "8.8.8.8", "city": "Mountain View", "country": "United States", "isp": "Google LLC", "timezone": "America/Los_Angeles", "lat": 37.4056, "lon": -122.0775}

Auto-Detect Caller's IP

curl https://api.gadgethumans.com/ip

MCP Tool: geolocate_ip

AI agents using the MCP server can call geolocate_ip. Available via Streamable HTTP endpoint or pip install gadgethumans-api-hub-mcp.

Response Data

  • ip: The IP address queried
  • city: City name
  • country: Country name
  • isp: Internet Service Provider
  • timezone: IANA timezone
  • lat/lon: Coordinates

Use Cases

  • Analytics and user segmentation
  • Geo-targeted content delivery
  • Fraud detection and security
  • AI agent context awareness
