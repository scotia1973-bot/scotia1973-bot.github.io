Why Use a Password Generator API?

AI agents and automated systems frequently need to generate secure credentials — for user accounts, API keys, database passwords, and temporary access tokens. The GadgetHumans Password Generator API provides cryptographically secure random password generation with 202,400+ configurations.

API Endpoint

GET https://api.gadgethumans.com/password

Parameters

ParameterTypeDefaultRangeDescription
lengthinteger164-256Password length
countinteger11-50Number of passwords
numbersbooleantruetrue/falseInclude digits 0-9
symbolsbooleantruetrue/falseInclude !@#$%^&*
uppercasebooleantruetrue/falseInclude A-Z
lowercasebooleantruetrue/falseInclude a-z

Example Usage

Generate 5 Strong Passwords

curl "https://api.gadgethumans.com/password?length=32&count=5&symbols=true&numbers=true"
# Response: {"passwords": ["...", "...", "...", "...", "..."], "count": 5, "length": 32}

Generate a PIN (Numbers Only)

curl "https://api.gadgethumans.com/password?length=6&numbers=true&symbols=false&uppercase=false&lowercase=false"

Security Notes

  • Passwords are generated server-side using cryptographically secure random number generators
  • No passwords are logged or stored — they exist only in the API response
  • HTTPS in transit ensures your generated passwords stay private

MCP Tool: generate_password

For AI agents using the MCP server, the generate_password tool provides the same functionality with the same parameters. Install via pip install gadgethumans-api-hub-mcp.
