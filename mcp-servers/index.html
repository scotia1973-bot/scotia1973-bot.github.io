🚀 How to Use MCP Servers

Prerequisites

Install uv (the Python package manager) for the easiest experience:

curl -LsSf https://astral.sh/uv/install.sh | sh

Claude Desktop

Open Claude Desktop → Settings → Developer → Edit Config, then paste the JSON config from the server card above into claude_desktop_config.json.

Cursor IDE

Create a .cursor/mcp.json file in your project root with the JSON from the Cursor tab above, or go to Cursor Settings → MCP → Add New.

Windsurf / Cline

Same config format as Cursor. Add the MCP server entry to your IDE's MCP settings with uvx gadgethumans-api-hub-mcp as the command.

Smithery.ai

Visit each server's Smithery page (linked from the server card) for one-click deployment to your agent.

From Python

Use the MCP client library to connect programmatically:

from mcp import ClientSession, StdioServerParameters
import asyncio

async def main():
    params = StdioServerParameters(
        command="uvx",
        args=["gadgethumans-api-hub-mcp"]
    )
    async with ClientSession(params) as session:
        tools = await session.list_tools()
        print(f"{len(tools.tools)} tools available")
        result = await session.call_tool("generate_password", {"length": 16})
        print(result.content[0].text)

asyncio.run(main())
