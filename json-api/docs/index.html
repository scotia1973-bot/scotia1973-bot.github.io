📋 JSON Prettify/Validate API Documentation

Base URL: https://api.gadgethumans.com/

Prettify, validate, or minify JSON strings with a simple HTTP GET request. Essential utility for debugging, formatting API responses, and ensuring JSON validity. No API key required for the free tier.

GET https://api.gadgethumans.com/json

Query Parameters

ParameterTypeDefaultDescription
jsonstringrequiredThe JSON string to process (URL-encoded)
modestringprettifyOperation mode: prettify, validate, or minify

Examples

Prettify JSON

curl "https://api.gadgethumans.com/json?json=%7B%22name%22%3A%22John%22%2C%22age%22%3A30%7D"
# {
#   "name": "John",
#   "age": 30
# }

Validate JSON

curl "https://api.gadgethumans.com/json?json=%7B%22valid%22%3Atrue%7D&mode=validate"
# {"valid":true,"message":"Valid JSON"}

curl "https://api.gadgethumans.com/json?json=not+json&mode=validate"
# {"valid":false,"error":"Unexpected token 'o', \"not json\" is not valid JSON"}

Minify JSON

curl "https://api.gadgethumans.com/json?json=%7B%22name%22%3A%20%22John%22%2C%20%22age%22%3A%2030%7D&mode=minify"
# {"name":"John","age":30}

JavaScript (Fetch)

const jsonString = JSON.stringify({ name: "John", age: 30 });

// Prettify
const pretty = await fetch('https://api.gadgethumans.com/json?json=' + encodeURIComponent(jsonString))
  .then(r => r.text());
console.log(pretty);

// Validate
const validation = await fetch('https://api.gadgethumans.com/json?json=' + encodeURIComponent(jsonString) + '&mode=validate')
  .then(r => r.json());
console.log(validation.valid); // true

Python

import requests, json

data = {"name": "John", "age": 30}

# Prettify
resp = requests.get('https://api.gadgethumans.com/json', params={
    'json': json.dumps(data),
    'mode': 'prettify'
})
print(resp.text)

# Validate
resp = requests.get('https://api.gadgethumans.com/json', params={
    'json': json.dumps(data),
    'mode': 'validate'
})
result = resp.json()
print(f"Valid: {result['valid']}")

Modes Explained

ModeResponseDescription

validateJSON objectReturns {"valid": true/false} with error details if invalid
minifyCompacted JSON stringRemoves all whitespace for smallest possible size
