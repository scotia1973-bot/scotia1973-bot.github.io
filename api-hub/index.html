🚀 Quick Start

No API key needed for the free tier. Just make a GET request:

# Generate a secure password
curl https://api.gadgethumans.com/password?length=16

# Create a QR code (saves as PNG)
curl https://api.gadgethumans.com/qr?text=https://example.com > qr.png

# Hash a string
curl https://api.gadgethumans.com/hash?text=hello&algorithm=sha256

# Generate a UUID
curl https://api.gadgethumans.com/uuid
