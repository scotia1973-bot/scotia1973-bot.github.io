🔤 Base64 Encode/Decode API Documentation

Base URL: https://api.gadgethumans.com/

Encode or decode text to and from Base64 format with a simple HTTP GET request. Useful for data transfer, API payloads, and debugging. No API key required for the free tier.

GET https://api.gadgethumans.com/base64

Query Parameters

ParameterTypeDefaultDescription
textstringrequiredThe text to encode or decode
modestringencodeOperation mode: encode or decode

Examples

Encode Text to Base64

curl "https://api.gadgethumans.com/base64?text=hello+world"
# aGVsbG8gd29ybGQ=

Decode Base64 to Text

curl "https://api.gadgethumans.com/base64?text=aGVsbG8gd29ybGQ=&mode=decode"
# hello world

Encode URL or Complex Strings

curl "https://api.gadgethumans.com/base64?text=https://example.com/api/v1/users"
# aHR0cHM6Ly9leGFtcGxlLmNvbS9hcGkvdjEvdXNlcnM=

JavaScript (Fetch)

// Encode
const encoded = await fetch('https://api.gadgethumans.com/base64?text=hello+world')
  .then(r => r.text());
console.log(encoded); // aGVsbG8gd29ybGQ=

// Decode
const decoded = await fetch('https://api.gadgethumans.com/base64?text=aGVsbG8gd29ybGQ=&mode=decode')
  .then(r => r.text());
console.log(decoded); // hello world

Python

import requests

# Encode
resp = requests.get('https://api.gadgethumans.com/base64', params={
    'text': 'hello world',
    'mode': 'encode'
})
print(resp.text)  # aGVsbG8gd29ybGQ=

# Decode
resp = requests.get('https://api.gadgethumans.com/base64', params={
    'text': 'aGVsbG8gd29ybGQ=',
    'mode': 'decode'
})
print(resp.text)  # hello world
