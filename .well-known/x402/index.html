🤖 x402 Agent Payment Protocol

GadgetHumans supports x402 — an HTTP 402 Payment Required protocol that lets AI agents pay for API access automatically via the Lightning Network.

💰 Lightning Wallet

Default wallet: bc1qxyz402gadgethumans
Agent payments are processed via Lightning invoices. 100 sats per 1,000 API requests.

🔌 How Agents Use It

When an API endpoint returns HTTP 402, the agent reads the X-402-Payment-Required header, pays the Lightning invoice, and retries with the proof of payment.

Example Response

HTTP/1.1 402 Payment Required
Content-Type: application/json
X-402-Payment-Required: lntb1u1p...invoice...

{
  "error": "payment_required",
  "message": "Free tier exceeded. Send 100 sats to continue.",
  "payment_url": "lightning:lntb1u1p...invoice..."
}
