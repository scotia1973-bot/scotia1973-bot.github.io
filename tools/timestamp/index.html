What is the Timestamp Converter API?

The GadgetHumans Timestamp Converter API is a free REST API that converts between Unix epoch timestamps and human-readable date formats. Whether you need to convert a Unix timestamp (like 1717000000) to ISO 8601, or parse a date string back to a Unix timestamp, this API handles it instantly with a single HTTP request.

The API accepts Unix timestamps in seconds since epoch, as well as various date string formats including ISO 8601 (2025-05-29T00:00:00Z), RFC 2822, and natural language dates. Output formats include iso, unix, rfc2822, and more. The free tier offers 100 requests per day with no API key required, making it perfect for developers debugging logs, working with database timestamps, building calendar applications, or processing time-series data from APIs that use epoch time.

⚡ Quick Start

Get started with a simple GET request. No authentication required.

cURL

# Quick example
# Convert Unix timestamp to ISO 8601
curl "https://api.gadgethumans.com/timestamp?value=1717000000&format=iso"

# Convert date string to Unix
curl "https://api.gadgethumans.com/timestamp?value=2025-05-29&format=unix"

Python

import requests

# Convert Unix timestamp to ISO date
url = "https://api.gadgethumans.com/timestamp"
params = {"value": "1717000000", "format": "iso"}
response = requests.get(url, params=params)
data = response.json()
print(f"Unix 1717000000 = {data["result"]}")

# Convert date to Unix
params = {"value": "2025-05-29", "format": "unix"}
response = requests.get(url, params=params)
data = response.json()
print(f"2025-05-29 = Unix {data["result"]}")

JavaScript (Fetch)

// Convert a timestamp from the browser
const timestamp = "1717000000";
fetch(`https://api.gadgethumans.com/timestamp?value=${timestamp}&format=iso`)
.then(res => res.json())
.then(data => {
console.log("ISO date:", data.result);
document.getElementById("tsResult").textContent = data.result;
});

❓ Frequently Asked Questions

What input formats are accepted?

Unix timestamps (seconds since epoch), ISO 8601 (2025-05-29), RFC 2822, and many common date formats.

Do I need an API key?

No! The free tier requires no API key or signup.

What output formats are supported?

iso (ISO 8601), unix (epoch seconds), rfc2822, and other standard date formats.

Does it handle timezones?

The API returns UTC by default. Specify timezone offsets in the input for localized conversions.

Can I convert milliseconds timestamps?

The API uses seconds since epoch. Divide milliseconds by 1000 before passing to the API.
