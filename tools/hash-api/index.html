⚡ Quick Start

Generate a Hash Generator API with a simple GET request. No authentication required.

cURL

# Generate a Hash Generator API and save as PNG
curl "https://api.gadgethumans.com/hash?text=https://example.com&format=png&size=300" > qr.png

# Get SVG output
curl "https://api.gadgethumans.com/hash?text=https://example.com&format=svg"

Python

import requests

# Generate Hash Generator API
url = "https://api.gadgethumans.com/hash"
params = {
  "text": "https://example.com",
  "format": "png",
  "size": "300",
  "color": "000000",
  "bgcolor": "FFFFFF"
}
response = requests.get(url, params=params)

# Save to file
with open("hash-api.png", "wb") as f:
  f.write(response.content)
print(f"Hash Generator API saved! Status: {response.status_code}")

❓ Frequently Asked Questions

Do I need an API key to use the Hash Generator API API?

No! The free tier requires no API key or signup. Just send a GET request and receive your Hash Generator API. For the Pro tier, you'll get a dedicated API key for higher limits.

What output formats are supported?

The API supports PNG (default), SVG (vector), and GIF formats. Use the format parameter to choose your preferred output.

Can I customize the colors of my Hash Generator API?

Yes! Use the color parameter for the foreground (default: black) and bgcolor for the background (default: white). Both accept hex values without the # symbol.

What is the maximum text length I can encode?

The API can encode up to 4,296 alphanumeric characters or 7,089 numeric characters, depending on the error correction level selected.

How many requests can I make on the free tier?

The free tier allows up to 100 requests per day per IP address. For higher limits, subscribe to the Pro plan for $5/month, which gives you 10,000 requests per day.
