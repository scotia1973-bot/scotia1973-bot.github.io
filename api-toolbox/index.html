Install in 30 Seconds

  1. Download the zip file above
  2. Unzip to a folder on your computer
  3. Open chrome://extensions in your browser
  4. Enable Developer Mode (toggle top-right)
  5. Click "Load unpacked" and select the folder
💡 The extension stays in developer mode and works exactly like a store-installed extension. Get updates from GitHub Releases.

What's Under the Hood

Every tool is powered by the Gadget Humans API Hub — 17 free REST APIs running on Cloudflare's global edge network. The extension caches results for 5 minutes and tracks daily usage (100 calls/day free).
