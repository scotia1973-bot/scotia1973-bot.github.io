🔑 Password Generator API Documentation

Base URL: https://api.gadgethumans.com/

Generate cryptographically secure random passwords with full control over character sets and length. No API key required for the free tier.

GET https://api.gadgethumans.com/password

Query Parameters

ParameterTypeDefaultDescription
lengthnumber16Password length (8-128 characters)
countnumber1Number of passwords to generate (1-20)
numbersbooleantrueInclude numeric digits (0-9)
symbolsbooleantrueInclude special characters (!@#$%^&* etc.)
uppercasebooleantrueInclude uppercase letters (A-Z)
lowercasebooleantrueInclude lowercase letters (a-z)

Examples

Generate a Strong 20-Character Password

curl "https://api.gadgethumans.com/password?length=20&numbers=true&symbols=true"

Generate 5 Passwords at Once

curl "https://api.gadgethumans.com/password?count=5&length=12"

Numbers & Uppercase Only (Easier to Read)

curl "https://api.gadgethumans.com/password?length=8&numbers=true&lowercase=false&symbols=false"

JavaScript (Fetch)

const response = await fetch('https://api.gadgethumans.com/password?length=16&count=3');
const passwords = await response.json();
console.log(passwords);
// ["aB3#kL9$xQ2@vR7&wP1", "Fg5*Ht8%Yn2#Jk4$Lp6", ...]

Python

import requests
resp = requests.get('https://api.gadgethumans.com/password', params={
    'length': 20,
    'numbers': 'true',
    'symbols': 'true',
    'count': 3
})
passwords = resp.json()
for p in passwords:
    print(p)

Response Format

// Single password (count=1)
"aB3#kL9$xQ2@vR7&wP1"

// Multiple passwords (count>1)
["aB3#kL9$xQ2@vR7&wP1", "Fg5*Ht8%Yn2#Jk4$Lp6", "Rt9#Xu4&Lp7$Qw2@Mn5"]

Headers

All responses include:

  • X-RateLimit-Limit — Daily request limit
  • X-RateLimit-Remaining — Remaining requests today
  • X-RateLimit-Reset — Unix timestamp when limit resets
