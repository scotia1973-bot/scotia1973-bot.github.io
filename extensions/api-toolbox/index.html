Frequently Asked Questions

Is this extension free?
Yes! The free tier includes all 8 tools with up to 100 API calls per tool per day. No signup required. No credit card. Just install and use.
Do I need an API key or account?
Nope. The extension works out of the box — no signup, no API key, no account creation. Just click and use any of the 8 tools.
What data do you collect?
Absolutely nothing. No browsing history, no personal info, no tracking, no analytics. Your theme preference and bookmarked results are stored locally on your device and never transmitted. Read our full privacy policy.
How do I use the side panel?
Right-click the extension icon and select "Open side panel", or pin the extension and press Ctrl+B (Cmd+B on Mac) to toggle the side panel. This gives you a larger workspace for the tools.
What happens when I hit the rate limit?
The free tier allows 100 API calls per tool per day. The footer shows your usage clearly. If you hit the limit, you can wait until the next day or upgrade to Pro ($5/mo) for unlimited calls.
Can I use this offline?
The extension requires an internet connection to make API calls to api.gadgethumans.com. However, your saved/bookmarked results are stored locally and viewable offline.
What permissions does it need and why?
storage — saves your theme preference and bookmarks locally. clipboardWrite — lets you copy results with one click. sidePanel — enables the Chrome side panel view. api.gadgethumans.com — required to make API calls for the tools.
