Privacy Policy

Last updated: May 12, 2026

✅ Zero Data Collection

Gadget Humans — API Toolbox collects absolutely no data. Period. No browsing history, no personal information, no analytics, no tracking scripts. Nothing.

What Data We Collect

None. The API Toolbox extension does not collect, transmit, or store any personal data, browsing history, or usage information.

💚

Anonymously Stored Locally

Your theme preference (dark/light) and any results you choose to bookmark are stored on your device using chrome.storage.local. This data never leaves your computer.

🚫

Never Collected

No browsing history, no personal information, no IP logging, no cookies, no analytics, no tracking pixels, no user accounts, no email addresses.

API Calls to api.gadgethumans.com

When you use any of the 8 tools, the extension makes an anonymous HTTP POST request to https://api.gadgethumans.com/ with the parameters you specify (e.g., text to generate a QR code, length for a password).

These API calls do not include any identifying information. The API server sees only the tool parameters and a generic rate-limit counter (no user ID, no IP stored).

Rate limiting is tracked using a hash of your IP address — the raw IP is never stored or logged. Rate limit data is automatically discarded after 24 hours.

How Data Is Stored

All extension data is stored exclusively in chrome.storage.local:

  • Theme preference — dark or light mode
  • Bookmarked results — any results you explicitly choose to save
  • API call history — a local log of your recent tool usage

This data is never synced to Google, transmitted over the network, or accessible to any third party. Clearing your browser storage or uninstalling the extension removes all data permanently.

Third-Party Services

The extension communicates with one third-party service:

  • api.gadgethumans.com — Our own API server that processes tool requests (QR codes, hashes, etc.). No personal data is transmitted. Standard server logs (IP, timestamp, requested endpoint) are retained for a maximum of 24 hours for rate-limiting purposes and are never shared.

No other third-party services, analytics platforms, advertising networks, or tracking services are used.

Data Sharing

We do not share any data. Since we collect no personal data, there is nothing to share. We do not sell, trade, or transfer any information to third parties.

Changes to This Policy

If this privacy policy changes, we will update the "Last updated" date at the top of this page. We encourage you to review this policy periodically. Material changes will be communicated via the extension's update notes.
